Tạo Use Case Diagram UML cho hệ thống **"Quản lý Quỹ Hỗ trợ Sinh viên TVU"**.

### Yêu cầu tổng quát

* Chỉ có duy nhất 1 Actor là **Kế toán**.
* Actor đặt bên trái hệ thống.
* Vẽ khung hệ thống với tên:

**HỆ THỐNG QUẢN LÝ QUỸ HỖ TRỢ SINH VIÊN TVU**

* Sử dụng ký hiệu UML chuẩn:

  * Actor (hình người)
  * Use Case (hình elip)
  * System Boundary (khung hệ thống)
* Thiết kế theo chiều dọc, rõ ràng, phù hợp đưa vào báo cáo đồ án.
